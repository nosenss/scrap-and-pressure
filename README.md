# Scrap & Pressure

Industrial rogue-lite compactor game for the browser. Pack scrap tight, press hard, draft upgrades, and climb the Top-10.

## Play

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173`.

## Build (itch.io zip)

```bash
npm run zip:itch
```
